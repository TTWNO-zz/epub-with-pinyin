# epub-pinyin
Adds Pinyin to .epub files

# Requirements
*<'https://github.com/TTWNO/dragonmapper'>
*BeautifulSoup4
*<'https://github.com/tsroten/pynlpir'>
*python(3) ebooklib

# Usage:

./add_pinyin.sh test.epub

New file will be saved as test-with-pinyin.epub

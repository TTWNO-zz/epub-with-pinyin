# epub-pinyin
Adds Pinyin to .epub files

# Requirements
* https://github.com/TTWNO/dragonmapper develop branch>
* python(3) beautifulsoup4
* python(3) jieba
* python(3) ebooklib

# Usage:

./add_pinyin.sh foobar.epub

New file will be saved as foobar-with-pinyin.epub
